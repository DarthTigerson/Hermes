Accesses
Onboarding Users with this role assigned will be able to add new employees.
Employee Updates Users with this role assigned will be able to change and update employee fields.
Offboarding Users with this role assigned will be able to offboard employees.
Manage Lists Users with this role assigned will be able to Add, Edit and Remove Lists (Departments, Sites, Contracts, Employers, Employment, Countries, Currency).
View Payroll Details Users with this role assigned will be able to Employees Payroll details.
Manage Admin Users with this role assigned will be able to Add, Edit and Remove Users, Roles and HR Teams.
API Reporting This role should be used for service accounts only. Users with this role assigned will be able to call an API request to Hermes to GET a report of all available users.
